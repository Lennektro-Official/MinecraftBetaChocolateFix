The Chocolate Pack is a mod for Minecraft Beta 1.7.3 with the aim to maintain it with all its oddities and stay as close as possible to vanilla, but fix some aspects that are somewhat broken.

Feature List

>  Skin Fix
>  Sound Fix
>  Fov Slider
>  Crash Slab Fix
>  Additional F3 Info
>  Old Cobble and Brick Texture
>  Quit Button always visible in main menu

This is a purley client side mod, so you can play with this on regular Minecraft Beta 1.7.3 servers. The provided download contains a zip file wich is a MultiMC/Prism Instance and another zip file labeled as a jarmod for those who want to include these fixes in other packs.
